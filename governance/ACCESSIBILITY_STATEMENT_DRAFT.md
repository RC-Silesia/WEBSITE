# Accessibility Statement Draft

Status: projekt.

Standard docelowy: WCAG 2.2 AA.

Strona RC Silesia jest obecnie publicznym stagingiem. Celem klubu jest rozwijanie strony zgodnie ze standardem WCAG 2.2 AA, ale pełna deklaracja zgodności nie jest jeszcze publikowana, ponieważ wymagany jest pełny audyt dostępności.

## Kontakt w sprawie barier

Bariery dostępności można zgłaszać pod adresem: [silesia@rotary.org.pl](mailto:silesia@rotary.org.pl).

## Status deklaracji

Pełna deklaracja dostępności zostanie opublikowana po audycie eksperckim i usunięciu barier wskazanych jako blokujące lub poważne.

## Elementy wymagające dalszej weryfikacji

- formularze;
- płatności;
- dokumenty PDF/DOCX;
- przyszły CMS;
- przyszły panel administracyjny.
