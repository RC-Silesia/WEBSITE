# Content Gaps Status Board - RC Silesia Website

Status: dokument roboczy dla Zarządu, Sekretarza i zespołu strony.

Strona RC Silesia działa jako publiczny staging demonstracyjny pod adresem:
https://rc-silesia.github.io/WEBSITE/

Strona nie jest jeszcze wersją produkcyjną. Przed zdjęciem `noindex/nofollow`
i zmianą `robots.txt` konieczne jest potwierdzenie danych, dokumentów oraz
decyzji organizacyjnych wskazanych poniżej.

## 1. Braki krytyczne

- Liczba członków zwyczajnych.
- Skład Komisji Rewizyjnej.
- Potwierdzenie Wiceprezydenta albo decyzja o usunięciu pola.
- Daty kadencji organów.
- Potwierdzenie publikacji danych osób funkcyjnych.

## 2. Braki wysokiego priorytetu

- Zatwierdzenie kalendarium.
- Uzupełnienie brakujących spotkań.
- Potwierdzenie dokumentów publicznych.
- Decyzja DOCX/PDF.
- Potwierdzenie numeru konta.
- Decyzja o module płatności.

## 3. Braki średniego priorytetu

- URL-e Instagram, X, YouTube, LinkedIn, JazzTV/media.
- Materiały medialne.
- Galeria zdjęć.
- Link do filmu ROTARY for PLANET.
- Partnerzy.
- Program młodzieżowy / Rotaract.

## 4. Braki zależne od decyzji strategicznych

- Domena produkcyjna.
- Zdjęcie `noindex/nofollow`.
- WordPress.
- ngOs.
- Panel członkowski.
- Płatności online.
- Newsletter mailingowy.

## 6. Rekomendowane działania

1. Sekretarz uzupełnia dane organów.
2. Zarząd potwierdza publikację danych osób funkcyjnych.
3. Osoba odpowiedzialna za stronę aktualizuje dokumenty po decyzji DOCX/PDF.
4. Skarbnik potwierdza konto i zakres modułu płatności.
5. Zespół komunikacji potwierdza social media, media i zdjęcia.
6. Po zatwierdzeniu treści można przygotować production readiness.

## 7. Uwagi końcowe

Strona może być pokazywana jako publiczna wersja demonstracyjna. Nie powinna
być traktowana jako produkcyjna do czasu uzupełnienia i zatwierdzenia danych
krytycznych oraz decyzji wskazanych w tym dokumencie.
